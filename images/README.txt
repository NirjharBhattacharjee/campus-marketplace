Put your item images in this folder.

When you add an item in the Firebase Console, set its "imageUrl" field to the
path of the file here, for example:

    imageUrl  ->  images/textbook.jpg

You can also use a full external image URL instead (e.g. an Unsplash link), in
which case you don't need to place a file here.

Tip: keep file names lowercase with no spaces (textbook.jpg, not "Text Book.JPG").
